# test-aventurier
## Lancement du programme
Ce programme prend en compte les arguments passés au lancement. Il est possible de le lancer soit en ligne de commande, soit en utilisant un IDE.
Dans les deux cas, il est nécessaire d'entrer les arguments dans l'ordre suivant : 
1. Chemin du fichier de la carte 
2. Chemin du fichier des coordonnées de l'aventurier
